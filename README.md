# 🧠 Desarrollo de una plataforma web para la gestión de pruebas psicológicas MoCA, dirigida al programa de Psicología de la Universidad Católica de Pereira.

Este repositorio contiene el código fuente del frontend de la plataforma, desarrollada para contribuir el proceso de evaluación del deterioro cognitivo mediante la prueba Montreal Cognitive Assessment (MoCA), en el programa de Psicología de la Universidad Católica de Pereira.

## 🌐 Descripción del proyecto

La plataforma web permite la digitalización de la prueba cognitiva, lo que posibilita su aplicación y almacenamiento por parte de los psicólogos. También ofrece acceso a resultados históricos, reduce errores humanos y contribuye a un entorno sostenible al eliminar el uso excesivo de papel.

## 🚀 Tecnologías utilizadas

- Angular (TypeScript)
- CSS
- Bootstrap
- JWT para autenticación
- Consumo de API REST (conectado al backend en Spring Boot)

## 🧩 Módulos clave

- **Evaluación**: Aplicación digital del test MoCA.
- **Seguimiento**: Visualización y análisis de resultados históricos.
- **Administrador**: Gestión de usuarios y configuración del sistema.
